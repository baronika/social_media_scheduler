# Social Media Scheduler

A web-based social media scheduling tool that allows users to plan, organize,
and schedule posts efficiently.

## Features
- Create and schedule social media posts
- User-friendly scheduling form
- Clean and responsive UI
- Built with React, Vite, Tailwind CSS

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
